# 冒泡排序

## 1.基本思想：

2个数比较大小，较大的数下沉，较小的数冒起来

## 2.算法描述

	(1)比较相邻的元素。如果第一个比第二个大，就交换它们

	(2)对每一对相邻元素作同样的工作，从开始第一对到结尾的最后一对，这样在最后的元素会是最大的数

	(3)针对所有元素重复以上步骤，除最后一个
	
	(4)重复1-3直至排序完成
